# Ptychozoon

A Python package for ptychography processing.

## Installation

It is recommended to install with qt dependencies:

```bash
pip install -e ".[qt]"
```

To install with no dependencies:

```bash
pip install -e .
```

To install the package for development:

```bash
pip install -e ".[dev, qt]"
```

## Usage

```python
import ptychozoon
```
