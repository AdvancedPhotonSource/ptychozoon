# Ptychozoon

A Python package for ptychography processing.

## Installation

```bash
pip install -e .
```

For development:

```bash
pip install -e ".[dev]"
```

## Usage

```python
import ptychozoon
```

## Development

To install in development mode with all dependencies:

```bash
pip install -e ".[dev]"
```

Run tests:

```bash
pytest
```
